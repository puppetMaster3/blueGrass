Can you try the Cloud API in 5 minutes to run in your enviroment? Lets get started:
 
Step 1: Edit an html file using your favorite set up and include this (cut/paste in): 
        

        <script src='http://f6.backendZero.com/acdn/libs/cloudAPI.js'></script>
        <script>
            console.log('oh hi')

            new CloudAPI().zipCodeUSA(94107, onData)
            // a look up city/state based on zipcode - a free/no-sign up service.

            function onData(data) {
                console.log(data)
            }
        </script>
Open the file in browser and view the javascript console (Tools/JavaScript Console). Congratulations, you just used Cloud API. It's just REST/Ajax that you don't have to set up. That was the hard part, ready for next step?
Step 2: Create an account on Cloud API by joining. You will get an app_key that you need. Then try this: 
        

        <script src='http://f6.backendZero.com/acdn/libs/cloudAPI.js'></script>
        <script>c = new CloudAPI('*YOUR APP KEY HERE*')></script>
        <script>
            var msg = new Object()
            msg.name = 'Jack'
            msg.score=  1232

            c.insert('names', msg, onPK) // insert()
            function onPK(data) {
                console.log(data)
            }
You just inserted a row by c.insert(...). 'names' is what you want to call the table, and msg proerties are dynamic column names. You can insert any columns you like, and create a table name dynamically. Can you figure out by yourself how to insert another row in another table?
ps: There is a helper method that creates the 'message' to send on a form in one line, tell you about it later.
Step 3: Try a select: 
        

        <script src='http://f6.backendZero.com/acdn/libs/cloudAPI.js'></script>
        <script>c = new CloudAPI('*YOUR APP KEY HERE*')></script>
        <script>
            var msg = new Object()
            msg.name = 'Jack'
            c.select('names', msg, onRet) // select()
            function onRet(data) {
                console.log(data.array_)
            }
You can test if the insert worked if you issue the c.select(...) and get some rows back. Do you think you can guess how to select from a different table other than 'names' and select diffrent query other than 'name=Jack'?
Your done for now, 5 minutes are up.
