What Is A Cloud API Anyway?

A Cloud API is a way for apps to access data in the cloud from somewhere else on the web. Think of this in terms of Facebook, in order for an application to get all of a users friends, they have to use the Facebook API to access that data and pull it into. We will work with a very simple API which is a service that has location data stored on zip code information.

Please note, this tutorial assumes a minimal understanding of javascript. If you have worked with API’s or are comfortable with .js, you may want to skip ahead.

So lets build an app where we want to collect a user's City, State and Zip. But for a better User Experience, we don’t want the user to have to enter any more fields than they have to. So if we only ask the user to manually enter a zip, then this API can look up the City and State fields, which we can grab to populate those fields.

In the following tutorial, I will show the newly added code in Bold

Step 1:

So lets start with a blank html file:

    <html>
    <head></head>
    <body>

    </body>
    </html>

Step 2:

Lets start by accessing the free API to pull in data. First we will add a library that will call such an API. Go ahead and copy this code into the body of your html doc.

    <html>
    <head></head>
    <body>
    <script src='http://ca_1.primusapi.com/acdn/libs/cloudAPI.js'></script>

    </body>
    </html>

This will download the library and let you use it.

Step 3:

Now lets create some code to use this library. We will use javascript to create an object that represents the new API, then a function to tell it what zip code to search for, and then finally another function to give us the data back.

    <html>
    <body>
    <script src='http://ca_1.primusapi.com/acdn/libs/cloudAPI.js'></script>
    <script>
        cApi =new CloudAPI()
        cApi.zipCodeUSA(94107, onData)

        function onData(data) {
            console.log(data)
        }
    </script>
    </body>
    </html>

You can run this code by opening it in browser. You won't see anything, so you'll have to 'view console'. On the browser menu select Tools/Javascript console or F12 or similar. Different browsers have different ways to show the console.

So lets look at what we did line by line:

    <script>

Because we are using javascript to create the API object and use functions, we need to run all this code within a <script></script> tag. 

    c=newCloudAPI()

We are creating a object called "cApi" (could be named whatever you like) as a new CloudAPI. This thing "CloudAPI" is defined in that library we downloaded in Step 2. It lets us handle and use information related to this API.

    cApi.zipCodeUSA(94107, onData)

So for the new object we created "cAPI" we are calling a function zipCodeUSA(). Again this function was defined in the library we downloaded in Step 2. The function takes 2 parameters. The first one is any zip code in USA. We are using 94107. And the second "onData" is a 'callback'. A callback allows us to do something like call another function, upon completion. This could be called anything you like.

    function onData(data) {
    console.log(data)
    }
    </script>

Here we are using the callback. So when we get the onData callback function, it runs the next function returning "data" as a parameter. This data is defined in the downloaded library, and has things like City, State, Coordinates, etc. Then we are finishing the script with the </script>

Step 4

Lets run the code! Open the html file in your browser, make sure to have your Console open. Your browser's console should look a bit like this output:

Object {Long: -122.39, Location: "NA-US-CA-SAN FRANCISCO", Lat: 37.76, ZipCodeType: "STANDARD", City: "SAN FRANCISCO"...}
